日本国内だけでなく世界的なトレンドも踏まえ、今後10年間で大きくバズる可能性が高い
作品案を30本提案してください。


# 要件
- 30本分の「タイトル + あらすじ」を作成すること
- 1つのテキストファイルに30本すべてをまとめること

# 出力フォーマット（各作品共通）
```
# タイトル名

## あらすじ
（2000文字程度）
```
- `sample.txt` は「フォーマットのみ」参考にすること
- `sample.txt` のタイトル内容・あらすじ内容は参考にしないこと
- `sample.txt` と被らない内容にすること
